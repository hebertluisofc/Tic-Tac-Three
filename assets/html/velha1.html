Velha #1 - Como jogar:

• O jogador X começa a partida.

• Os jogadores alternam turnos marcando uma das 9 células.

• Ganha quem formar uma linha com 3 símbolos iguais:

  • linha horizontal
  • linha vertical
  • diagonal

• Se todas as células forem preenchidas e ninguém completar uma linha, o jogo termina em velha.
